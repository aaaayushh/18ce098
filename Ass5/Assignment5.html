NATIONAL WORKSHOP ON WEB TECHNOLOGY

Join us on Saturday afternoons for a very hands on, very project oriented exploration of internet technologies. Sessions run every Saturday afternoon from 3-4:30pm. You can register for individual sessions or for a discounted six-week package. Participants can start anytime they want, sessions do not have to be consecutive.

New students will be introduced to the current group class project and brought up to speed at their own pace. When they are ready and interested they will have the opportunity to do an individual or partner project at the same time. They will build real, interactive web pages from the ground up using current internet development techniques and interface design.
